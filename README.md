# Program-SimpleCashier
Manage product list and payment feature with output payment history with mysql database local root folder no password needed

Tech Stack :
- PHP Native
- Tailwind CSS CDN version (so make sure you're always online)
- HTML, JAVASCRIPT

How To Use ? :
1.) Install localhost server app like XAMPP or Laragon
2.) Open Browser and Import The SQL file at localhost/phpmyadmin
